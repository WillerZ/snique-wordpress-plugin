== Description ==

snique embeds encrypted messages in your blog in such a way that their presence is undetectable. Users with a snique client [for iOS](https://github.com/WillerZ/snique-client-ios "snique client for iOS on github") or [for android](https://github.com/WillerZ/snique-client-android "snique client for android on github") are able to read them if you and they are sharing the same secret key.

Development of the plugin is [on github](https://github.com/nomzit-dot-com/snique-wordpress-plugin "snique on github").

== Installation ==

Follow the instructions for your version of Wordpress.
